Types of Digital Marketing - Interactive Web App
This is an interactive web application that visually demonstrates the various types of digital marketing. The app features an animated background, clickable topic bubbles, an info card for each topic, and a quiz section to reinforce learning. The design is responsive and visually engaging, making it ideal for educational or presentation purposes.

Features
Animated Background: Dynamic floating bubbles and morphing shapes create an immersive experience.

Interactive Topic Bubbles: Click on each bubble to learn about a specific type of digital marketing.

Info Cards: Each topic has a detailed card with icons, titles, and bullet points.

Big 3D Card: Displays in-depth information with an image and bullet list for each topic.

Quiz Section: Test your knowledge with multiple-choice questions after exploring all topics.

How It Works
On load, the main bubble drops into view with animated floating icons.

Click the main bubble to reveal topic bubbles arranged in a circle.

Click any topic bubble to open its info card and big 3D card.

After clicking all topic bubbles, a quiz bubble appears.

Click the quiz bubble to start the quiz and test your knowledge.

Getting Started
Prerequisites
A modern web browser (Chrome, Firefox, Edge, etc.)

Running the App
Clone this repository to your local machine.

Open index.html in your browser.

Explore the topics and take the quiz.

Customization
Replace image paths in the localImages object with your own images.

Add or modify quiz questions in the quizQuestions array.

Customize colors and animations in the CSS section.
